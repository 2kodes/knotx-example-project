# Running the example project

## Prerequisites
- JDK 8+

## Getting started
- Download distro
- Unpack distro
- Replace configs
- Resolve dependencies
```
bin/knotx resolve
```
- Run knotx
```
bin/knotx run-knotx
```


## Running the project
Once you downloaded the Knot.x and added your configurations, just run:
```
bin/knotx run-knotx
```
